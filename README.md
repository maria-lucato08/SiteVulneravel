Comandos do git para a branch
```
git branch == mostra em qual branch você está
git checkout -b '.....' == sai da branch em que você está e cria a nova branch(você da o nome da branch conforme o que você for fazer. ex: git checkout -b 'testando-criacao-branch' sem ponuações)
git checkout 'testando-criacao-branch' == mudar para uma branch que ja existe(dentro das '' vc passa o nome da branch. ex: main que e a princial)
